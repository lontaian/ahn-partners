⚙️ 시뮬레이션 기능

일간 목표 설정

  • 슬라이더 또는 입력란에 일일 목표 충전량 입력 (100kg 단위)
  • 월간 목표 자동 계산 (일간 × 30일)
  • MTOP 충족률 실시간 업데이트

MTOP 충족률 계산

  • 예상 충전량 = 반경 내 거래처 월간 충전량 합계
  • 충족률 = (예상 충전량 ÷ MTOP 목표) × 100%
  • 필요 버스 대수 = 부족량 ÷ 247.86kg (평균 월간 충전량/대)

MTOP 상태 메시지

충족률상태권장 조치
≥100%🟢 목표 달성안정적 수익 구조 확보
70~99%🔵 달성 근접추가 영업으로 달성 가능 (약 N대)
40~69%🟡 영업 필요적극적 거래처 영업 필요
<40%🔴 발굴 우선신규 거래처 발굴 / MTOP 단계 재검토

반경 조절 시 해당 범위 내 거래처만 테이블에 표시:

  • 거래처명, 수소버스 대수, 충전량, 도로거리
💡 영업 목표: 2026년 10월 오픈까지 충분한 수소버스 고객 확보가 핵심

📈 차트 분석

법인별 충전량 (도넛 차트)

  • 차트 내부에 법인명 및 비율(%) 직접 표시
  • 마우스 호버 시 충전량(kg) 및 정확한 비율 툴팁 표시
  • 온버스: 주력 법인 (약 83% 비중)
  • 원더온: 중형 법인 (약 10%)
  • 알이백: 소형 법인 (약 7%)
  • 월별 전환 시 차트 데이터 자동 갱신
  • 하단 테이블에서 거래처수/버스/충전량/횟수/일평균/충전주기 상세 확인

거래처별 충전량 (막대 그래프)

  • 상위 5개 / 10개 / 전체 선택 가능
  • 충전량 기준 내림차순 정렬
  • 막대 라벨에 버스대수, 충전횟수 함께 표시
  • 마우스 호버 시 상세 수치 표시
  • 모바일: 동적 높이 조절로 최적화 표시
💡 인사이트: 삼성전자(수원)이 전체 충전량의 약 36% 차지

🏢 현재 거래처 탭

원더모빌리티 소속 거래처 상세 (10월: 16개, 11월: 15개, 12월: 15개, 1월: 15개):

  • 수소버스 대수, 월간 충전량, 충전횟수
  • 충전금액(VAT), 유가보조금 - 실제 청구 금액
  • 1회 충전량, 도로거리 및 이동시간
  • 포곡 거리: 포곡 충전소까지 도로거리
  • 기존충전소 평균: 해당 월 이용 충전소들의 가중평균 거리

  • 11월 전용: 전월대비(kg), 증감률(%) 컬럼 추가
  • 반경 내 거래처 녹색 하이라이트
  • 고객 이탈: DL E&C (11월 충전 없음)
💡 마우스 호버: 거래처 행에 마우스를 올리면 해당 거래처가 이용한 충전소별 충전량과 거리 정보가 툴팁으로 표시됩니다.
