AI 자동화 / API 연동 — ChatGPT·Claude 등 AI API 연동, 반복 업무 자동화, AI 챗봇 구축. 외부 API(PG사, 공공데이터 등) 연동 개발도 가능합니다.

자주 묻는 질문

어떤 종류의 개발을 의뢰할 수 있나요?

웹앱, 모바일 앱(iOS/Android), 업무 관리시스템(ERP), 대시보드, AI 자동화, GIS 지도 시스템 등 다양한 개발이 가능합니다. 실제로 431대 차량을 관리하는 ERP, 1,556개 노선 최적화 시스템, 누적 100만+ 다운로드 앱 등을 구축한 경험이 있습니다.

개발 기간은 보통 얼마나 걸리나요?

프로젝트 규모에 따라 다르지만, 일반적으로 MVP는 2~4주, 풀 기능 시스템은 4~8주 소요됩니다. 운송 ERP(FMS)의 경우 요구사항 분석부터 프로덕션 배포까지 4주 만에 완료했습니다. 1주 단위로 동작하는 데모를 공유하며 진행합니다.

기획서가 없어도 의뢰할 수 있나요?

네, 가능합니다. 아이디어 단계부터 함께합니다. 구두 설명, 엑셀 파일, 참고 사이트 등 어떤 형태의 요구사항이든 정리해 드립니다. 실제로 FMS 프로젝트는 18개 엑셀 파일과 현업 인터뷰를 분석해 요구사항을 직접 도출했습니다.

주로 어떤 기술 스택을 사용하나요?

프론트엔드는 Next.js, React, TypeScript, Tailwind CSS를 주로 사용합니다. 백엔드는 Python(FastAPI), Node.js이며, 데이터베이스는 PostgreSQL, MySQL을 사용합니다. 지도 시스템은 MapLibre GL, Leaflet, Kakao Maps를 활용합니다.

모바일 앱도 개발 가능한가요?

네, iOS와 Android 모바일 앱 개발이 가능합니다. 기획부터 스토어 등록, 운영까지 전 과정을 지원합니다. 누적 100만+ 다운로드를 달성한 앱 출시 경험이 있으며, 웹서비스와 앱을 동시에 구축하는 것도 가능합니다.

결과물을 미리 확인할 수 있나요?

네, 5개 이상의 라이브 데모를 제공하고 있어 실제 동작하는 결과물을 직접 확인하실 수 있습니다. 운송 ERP, 노선 편집기, 대시보드 등의 데모가 준비되어 있습니다. 개발 중에도 1주 단위로 동작하는 데모를 공유하며 소통합니다.
