모바일 앱 (iOS / Android) — 기획부터 스토어 등록·운영까지. 누적 100만+ 다운로드 앱 출시 경험을 바탕으로, 웹서비스와 앱을 동시에 구축하는 것도 가능합니다.

AI 자동화 / API 연동 — ChatGPT·Claude 등 AI API 연동, 반복 업무 자동화, AI 챗봇 구축. 외부 API(PG사, 공공데이터 등) 연동 개발도 가능합니다.
